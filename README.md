# Portfolio Website - Nguyễn Thị Quỳnh Nga

Một website portfolio cá nhân hiện đại và sáng tạo dành cho sinh viên ngành CNTT - Đồ họa kỹ thuật số.

## 🎨 Phong cách thiết kế

- **Tông màu**: Nâu vàng ấm áp (warm brown + golden beige)
- **Phong cách**: Hiện đại, tối giản, ấm áp, sáng tạo
- **Cảm giác**: Thân thiện, nghệ thuật, mềm mại nhưng chuyên nghiệp
- **Font chữ**: Poppins (hiện đại, dễ đọc)
- **Responsive**: Hoàn toàn trên desktop, tablet và mobile

## ✨ Tính năng nổi bật

- **Background động**: Animation chó mèo đung đưa nhẹ nhàng
- **Glassmorphism**: Hiệu ứng kính mờ hiện đại
- **Theme toggle**: Chuyển đổi Light/Dark mode
- **Floating particles**: Hạt bụi nổi nhẹ
- **Cursor glow effect**: Hiệu ứng glow theo con trỏ chuột
- **Scroll progress bar**: Thanh tiến trình cuộn trang
- **Loading animation**: Animation tải trang
- **Fade-in animation**: Hiệu ứng xuất hiện khi cuộn
- **Smooth scrolling**: Cuộn mượt mà
- **Navbar sticky**: Thanh điều hướng dính khi cuộn

## 🚀 Cách chạy

1. Clone repository:
```bash
git clone https://github.com/ngaq16210-star/mynewpageq123.git
cd mynewpageq123
```

2. Mở file `index.html` trong trình duyệt hoặc sử dụng server local:
```bash
python3 -m http.server 8000
```

3. Truy cập `http://localhost:8000`
